Проект WordFormCheckerApp

Данный проект производит поиск ошибок словообразования в предложении на английском языке.

Программа является консольным приложением, которое на вход принимает 3 аргумента argv[1] argv[2] argv[3]. 
- argv[1] - Путь к некорректному файлу - 1 строка с предложением с возможными ошибками.
- argv[2] - Путь к корректному файлу - 2 строки: 1 строка - корректное предложение, 2 строка-pos-tags
- argv[3] - Путь к выходному файлу, куда будет записан результат работы программы

В корневой директории распологаются все .cpp и .h файлы вместе с файлами тестов.

В директории Doxy_files содержится документация в стиле doxygen для данной программы 
Просмотреть её можно открыв любой файл с расширением .html, затем перемещаясь по страницам документации.

В директории release распологается exe файл программы, bat_tests.bat - необходима для запуска тестов и директория tests, в которой содержатся все тесты и сами результаты тестов.
